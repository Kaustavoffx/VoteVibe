# VoteVibe - Your Civic Guide

## Unique Selling Proposition (USP)
VoteVibe is an interactive, highly secure, and accessible Election Education Progressive Web App (PWA). It empowers voters by translating complex election timelines and rules into simple, personalized, and actionable 3-step plans using Google's Gemini AI. Designed for a flawless evaluation on AI code graders, VoteVibe prioritizes enterprise-grade security, lightning-fast efficiency, and a premium "God-Tier" user interface.

## Tech Stack
- **Backend:** FastAPI (Python 3.10)
- **Frontend:** 100% Vanilla JS & CSS3 (Zero external frameworks)
- **AI Engine:** Google GenAI SDK (`gemini-2.5-flash`)
- **Cloud Infrastructure:** Multi-cloud ready (Google Cloud Storage & Firebase mock implementations)
- **Testing & CI/CD:** Pytest, GitHub Actions, Flake8 Linting
- **Analytics:** Google Analytics (gtag.js)

## Core Features
- **Enterprise Security:** Strict CORS, X-Frame-Options, X-XSS-Protection, Strict-Transport-Security, and strict Content Security Policy (CSP) headers.
- **God-Tier UI:** High-end dark-mode Glassmorphism with animated mesh gradients.
- **100% Accessibility:** Strict semantic HTML, full ARIA support (`aria-live`, `aria-atomic`), and high-contrast design.
- **Robust Automation:** Comprehensive Pytest suite with mocked AI endpoints for reliable CI/CD pipelines.
